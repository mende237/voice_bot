le script python AGI_example_pg est le script python donnee dans le guide Astericx officiel 
permettant de controler un plan d'appel ( en anglais Dialplan ) d'asterix par des commandes
utilisant les flux "stdin" et "stdout" 

Le but de ce script permet de definir un plan d'appel qui permettra suivant une numero de compte 
d'un utilisateur astrisk definit dans sip.conf ( un appelant ), a fournir ses biens (argent) dis-
ponibles dans son compte
